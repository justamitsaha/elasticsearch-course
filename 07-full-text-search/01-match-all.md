## Match All

- Used to retrieve all documents in an index.
- Similar to `SELECT * FROM [table]` in SQL.

### Demo

```
# delete index
DELETE /products

# create index with the mapping
PUT /products
{
  "mappings": {
    "properties": {
      "name": { "type" : "text"},
      "size":  { "type" : "keyword"},
      "color":  { "type" : "keyword"},
      "brand":  { "type" : "keyword"}
    }
  }
}

# insert records
POST /products/_bulk
```
